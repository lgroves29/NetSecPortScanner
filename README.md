# NetSecPortScanner
Final project for Network Security implementing a port scanning tool


## Introduction

## Requirements
This tool uses the following python3 modules:
* scapy
* socket
* random
* argparse
* time

## To run:

This tool runs from the command line with the following options:
* mode [normal/syn/fin] 
* order [order/random] 
* ports [all/known]
